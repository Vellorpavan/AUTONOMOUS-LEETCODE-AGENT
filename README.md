## рҹ§  What Is This?

<table>
<tr>
<td width="60%">

**LeetCode Autonomous Agent** is a fully automated AI system that logs into your LeetCode account and solves **every single problem** вҖ” on its own, without any human input after you press start.

It combines three things:

1. **A custom MCP Server** (Model Context Protocol) built from scratch вҖ” wraps LeetCode's private GraphQL API as clean, callable tools
2. **Claude Opus 4** as the AI brain вҖ” reads problems, reasons about algorithms, writes optimal Python 3 code
3. **An autonomous agent loop** вҖ” manages the problem queue, submits solutions, reads verdicts, fixes failures, and moves on вҖ” forever

You start it once. It runs until your LeetCode profile is solved.

<div align="center">

## рҹӨ– Agent System Prompt

The agent uses an elite 5-part system prompt:

```
PART 1 вҖ” IDENTITY
  Elite competitive programmer. 10+ years ICPC + LeetCode.
  Never guesses. Verifies everything.

PART 2 вҖ” AGENT LOOP
  Startup вҶ’ auth check вҶ’ load queue вҶ’ sort Easy/Medium/Hard
  Per problem: skip check вҶ’ solve вҶ’ submit вҶ’ check вҶ’ branch on verdict
  Rate limit guard (wait 60s on 429) вҶ’ session log вҶ’ next problem
```

## рҹҗӣ Troubleshooting

<details>
<summary><b>вқҢ "Not authenticated" error</b></summary>

Your cookies have expired (they last 2вҖ“4 weeks).

1. Log into leetcode.com in your browser
2. Get fresh cookies (see [Getting Your LeetCode Cookies](#-getting-your-leetcode-cookies))
3. Update `agent/.env` with the new values
4. Restart the agent

</details>

<details>
<summary><b>вқҢ "MCP server not found" / "Cannot find module"</b></summary>

The TypeScript hasn't been compiled yet.

```bash
cd mcp-server
npm install
npm run build
```

</details>

<details>
<summary><b>вқҢ Agent stops after one problem</b></summary>

Check `agent/logs/` for the latest session file. Look for the error. Common causes:

- API rate limit hit вҶ’ add `time.sleep(2)` between problems
- Network timeout вҶ’ check internet connection
- Wrong MCP path вҶ’ check `MCP_SERVER_PATH` in `.env`

</details>

<details>
<summary><b>вқҢ "Premium problem" errors</b></summary>

The agent automatically skips premium-only problems. If you have LeetCode Premium, the cookies will already include premium access вҖ” no extra config needed.

</details>

<details>
<summary><b>вқҢ Python venv issues</b></summary>

```bash
cd agent
rm -rf venv
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

</details>
